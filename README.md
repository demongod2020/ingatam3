## 1. 근대적 개인과 정치적 딜레마
* [cite_start]**근대적 인간관**: 인간을 태어날 때부터 자유롭고 평등한 권리를 가진 독립적인 존재로 바라보기 시작했습니다[cite: 6].
* [cite_start]**핵심 문제**: 모든 개인이 자유롭다면 어떻게 공동체의 질서를 유지할 것인가, 즉 **개인의 기본권과 공동체의 이익을 어떻게 양립시킬 것인가**가 정치철학의 주요 과제가 되었습니다[cite: 8, 9, 11].

## 2. 존 로크의 사상: 자연법과 동의에 의한 정부
* [cite_start]**자연상태**: 타인의 의지에 종속되지 않고 자연법의 테두리 안에서 자신의 행동과 소유물을 결정할 수 있는 **완벽한 자유와 평등의 상태**입니다[cite: 17, 18, 19].
* [cite_start]**자연법의 제한**: 자유의 상태이지만 방종은 아니며, 이성(자연법)에 따라 타인의 생명, 건강, 자유, 재산에 피해를 주어서는 안 됩니다[cite: 25, 28].
* [cite_start]**사회 형성의 이유**: 재산을 안전하게 보호하고 외부 위협으로부터 평화로운 삶을 누리기 위해 개인들이 **합의(동의)**함으로써 공동체를 결성합니다[cite: 53, 67].
* [cite_start]**정치적 자유**: 국가의 입법권 아래 존재하는 자유로, 자의적인 의지가 아닌 사회 구성원 모두에게 적용되는 **일정한 규칙(법)에 따르는 자유**를 의미합니다[cite: 37, 39, 40].

## 3. 장자크 루소의 사상: 사회협약과 일반의지
* [cite_start]**사회계약의 필요성**: 자연상태에서 생존을 방해하는 장애물이 개인의 힘을 능가할 때, 인류는 생존을 위해 **힘의 총화**를 이루어야 합니다[cite: 80, 81, 83].
* [cite_start]**전적인 양도**: 각 개인은 자신의 모든 권리를 공동체 전체에 양도합니다[cite: 94]. [cite_start]이를 통해 모두가 평등해지며, 자신을 전체에 양도함으로써 결과적으로는 누구에게도 예속되지 않는 상태가 됩니다[cite: 95, 99, 100].
* [cite_start]**일반의지(General Will)**: 각 개인은 자신의 신체와 능력을 **일반의지의 최고 지도** 아래 둡니다[cite: 102]. [cite_start]이 결합을 통해 탄생한 공적 인격을 '공화국' 또는 '정치체'라고 부릅니다[cite: 105].
* [cite_start]**도덕적 자유**: 사회상태로의 이행은 본능을 정의와 이성으로 대체하며, 인간에게 **도덕성**을 부여합니다[cite: 113, 114]. [cite_start]루소는 스스로 규정한 법률에 복종하는 것을 참된 '도덕적 자유'라고 정의했습니다[cite: 125].
